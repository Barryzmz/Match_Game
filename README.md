# 🧠 Match Game - 動物配對小遊戲

這是一個使用 `.NET WPF` 開發的桌面應用程式小遊戲，目標是找出相同的動物 emoji 並完成配對。當你成功配對兩個相同的圖案，它們將會消失。考驗你的記憶力與觀察力！

---

## 🎮 遊戲特色

- 使用 emoji 作為圖像（無需額外圖片資源）
- 每回合都會隨機打亂排列
- 成功配對會隱藏圖案
- 可設定計時器，追蹤完成時間
- 使用 `TextBlock` 和 `Grid` 排版，簡潔直觀
